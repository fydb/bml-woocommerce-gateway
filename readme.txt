An unofficial payment gateway integration for accepting Bank of Maldives Connect payments in WooCommerce.

== Description ==

IMPORTANT DISCLAIMER: This is NOT an official Bank of Maldives plugin. This plugin is an independent integration and is not affiliated with, officially connected to, or endorsed by Bank of Maldives PLC. All related trademarks and copyrights belong to their respective owners.

This unofficial plugin adds Bank of Maldives Connect as a payment method to WooCommerce. It has been developed independently to help WooCommerce store owners integrate with Bank of Maldives payment services.

Features:

* Accept payments directly on your store
* Test mode for development and testing
* Support for both test and live API keys

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bank-of-maldives-gateway` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > Settings > Payments and configure the Bank of Maldives payment gateway
4. Enter your API credentials
